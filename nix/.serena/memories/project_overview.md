# Project Overview

This repository subtree contains a modular NixOS and Home Manager configuration tailored for WSL and NixOS hosts. The Nix project root is `/home/ukasha/code/.dotfiles/nix`, while the enclosing Git repository root is `/home/ukasha/code/.dotfiles`.

The configuration is flake-based and uses `flake-parts`, Home Manager as a NixOS module, and a custom dendritic module structure based on automatic path scanning. The project documents architecture and current decisions in `AGENTS.md`, `CONTEXT.md`, and the OKF bundle under `okf/`.
