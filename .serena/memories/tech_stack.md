# Tech Stack

## Core Technologies

- **Nix**: Package manager (unstable channel).
- **Flakes**: Dependency management and system entry points.
- **flake-parts**: Modular flake output organization.
- **Home Manager**: User-level configuration management.
- **nixos-wsl**: WSL2 integration for NixOS.

## Tooling & CLI

- **nh (nix-helper)**: Primary tool for system/home activation and cleanup.
- **treefmt-nix**: Unified formatting interface.
- **direnv / nix-direnv**: Per-directory development environment activation.
- **ghostty**: Default terminal.
- **helix (hx)**: Default editor.
- **nushell**: Primary interactive shell.
- **yazi**: Terminal file manager.
- **zellij**: Terminal multiplexer.

## Styling & Themes

- **Catppuccin**: Global theme flavor (mocha) applied via `catppuccin/nix`.
- **Plasma Manager**: KDE Plasma configuration via Home Manager.
